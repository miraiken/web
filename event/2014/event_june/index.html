６月開催　みらい研究室 ～科学へのトビラ～

毎年６月にお台場の日本科学未来館で開催される、
みらい研究室実行委員会主催イベント
「科学へのトビラ」２０１４年も開催決定！！

日時： 6/7、8　(10：00～17：00)
場所： 日本科学未来館　(1Ｆ、7Ｆ)

子供と大人が一緒に楽しめる企画が充実！
ぜひ保護者の方と一緒にお越しください！
みらい研究室実行委員一同、心よりお待ちしております。

イベント当日の諸注意

混雑時、会場内の安全を確保するために入場規制をかけることがあります。ご了承ください。
