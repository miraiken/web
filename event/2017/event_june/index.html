6月開催　みらい研究室 ～科学へのトビラ～

みらい研究室実行委員会主催イベント
「科学へのトビラ」2017年も開催決定！！

みらい研究室 ～科学へのトビラ～について

小中学生に科学の楽しさを伝えたいという思いを胸に、
東京理科大学の学生が集まり2006年から毎年開催している科学体験イベントです。
今年は、東芝未来科学館で初開催！！

日時

日時： 6/10, 11　(10：00～17：00)
（受付終了 16：30）
場所： 東芝未来科学館　(2Ｆ、3Ｆ)

子供と大人が一緒に楽しめる企画が充実！
ぜひ保護者の方と一緒にお越しください！
みらい研究室実行委員一同、心よりお待ちしております。

  • 整理券を配布しているブースがございます。整理券は先着順となりますのでお早めにお越しください。
