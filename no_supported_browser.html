お使いのブラウザはサポートされません(We will never support your browser)

Cause: unknown

お使いのブラウザは古すぎます。新しいブラウザを利用してください

推奨するブラウザの一例です。これ以外のブラウザも対応している場合はあります

  • Google Chrome
  • Firefox
  • Microsoft Edge

みらい研究室実行委員会では、セキュリティを考え、古いブラウザで閲覧しようとした場合、このページヘリダイレクトするようにしています。
例えばInternetExplorer10は古いブラウザとみなされこのページヘリダイレクトされます。
古いブラウザを使い続けることは、単にHTML5やECMAScript5への対応が不十分で表示が崩壊するだけではなく、お使いの端末を危険に晒します。
みらい研究室実行委員会ではInternetExplorer自体の利用を推奨していません。これはInternetExplorerでの動作確認はIE11のみで行なっているからです。
Windows　Vista以前を利用しているユーザーはアクセス解析の結果わずか1.5%しかいないことがアクセス解析より明らかになっています。このことからみらい研究室実行委員会ではこの1.5%を切り捨てることは妥当だと考えています。
2017年2月現在、MicrosoftがサポートするすべてのWindowsOSでは少なくともInternetExplorer11を利用できます。にもかかわらずそれより古いバージョンのInternetExplorerをサポートすることはユーザーのPCを危険に晒し、また悪意のあるクラッカーを手助けすることになります。
したがってInternetExplorerはバージョン11を除いてサポートしません。

みらい研究室実行委員会はWindows200/XP/Vistaからの利用を推奨しません

Windows200/XP/VistaはすでにMicrosoftからのサポートが終了されています。
Windows7以降で見つかったバグがWindows200/XP/Vistaにも存在する可能性が高く、
修正バッチが配布されないこれらの環境を利用し続けることは、あなたのPCを危険に晒すだけではなく、他のPCへの攻撃の踏み台にされる危険があります。

Internet ExplolorはOSのサポート終了と同時にサポートが終了しています。
したがって、使用を中止することを強く推奨します。

Google Chromeは2016年3月31日にすでにWindowsXP/Vista向けのサポートを終了しています。
Google Chrome Blog: Updates to Chrome platform support
したがって、使用を中止することを強く推奨します。

FirefoxはVersion52がWindowsXP/Vista向けの最後の更新となります。
重要: Firefox は Windows XP および Vista のサポートを終了します | Firefox ヘルプ

OperaはWindowsXP/Vista向けのサポートが当面継続することが見込まれるごく僅かなブラウザの一つです。
Opera 36がWindowsXP/Vista向けの最後のバージョンになります。
Opera developer 37 最初のリリース - Opera Japan
Windows2000向けのOpera 12のサポートも継続しています。
Opera 12 and Opera Mail security update - Opera Security
しかしながら、みらい研究室実行委員会ではOperaでの正常な閲覧を確認していません。

Android/iOSをご利用の方へ

みらい研究室実行委員会では、iOS7以前および、Android4.0.x以前、およびすべてのAndroid標準ブラウザ(Chromeのことではありません)をサポートしません。
iOS7以前および、Android4.0.x以前をご利用の方は新しいOSを利用してください。
Android4.1以降をご利用でAndroid標準ブラウザを使用している場合はChromeやFirefoxをご利用ください。

サポートされているブラウザを利用しているにもかかわらずこのページへリダイレクトされた場合は
https://github.com/miraiken/web/issues
までお知らせください。
